## What is KDONTOP?

KDONTOP is an **invite-only community hub** that combines **GitHub-like code repositories** with a **full-featured forum** — wrapped in a sleek midnight dark theme. No big-tech data harvesting. No distractions. Just your team, your code, and your conversations.

> Think of it as **GitHub + Discord + a Forum**, privately hosted and fully under your control.

---

## ✨ Features

### 🏠 Personalized Dashboard

- **Time-aware greeting** — "Good morning", "Good afternoon", "Good evening" with your name
- **Recent Repositories** and **Recent Threads** side-by-side so you never miss anything
- **Quick action cards** — Create a repo, start a thread, or explore in one click
- Clean, card-based layout designed for focus

### 📁 Repositories

- **Create & manage** code repositories with descriptions, tags, and visibility controls
- **Upload files** individually or as ZIP archives — extracted and browsed in-app
- **Fine-grained access control** — Owner, Maintainer, Contributor, Viewer roles per repo
- **Issues** — Track bugs and feature requests with labels and assignments
- **Discussions** — Threaded conversations tied to each repository
- **Releases** — Version and distribute your project builds
- **Visibility levels** — Private, Unlisted, or Public per repository

### 💬 Forum

- **Organized by categories** — Announcements, Off-topic, and any custom categories admins create
- **Role-restricted categories** — Admins can lock categories to specific roles (e.g., "VIP Lounge" only for VIP members)
- **Rich threads** — Full Markdown support with live preview, syntax highlighting, and a formatting toolbar
- **Comments** with nested discussions, upvotes, and view counts
- **Pin & Lock** — Moderators can pin important threads and lock heated ones
- **@Mentions** — Tag any user with `@username` and they get notified instantly
- **Emoji Reactions** — Discord-style reaction picker with 12 emojis on threads and comments

### 🏷️ Custom Roles (Discord-style)

- **Colored role badges** displayed next to usernames everywhere — threads, comments, nav bar
- **Default roles** — New members auto-assigned "Freshy" 🌱 on signup
- **Admin-managed** — Create unlimited roles with custom names, colors, emojis, and permission levels
- **Role hierarchy** — User → Trusted → Moderator → Admin permission tiers
- **Category access control** — Restrict which roles can see specific forum categories

### 🔔 Notifications

- **Real-time notification bell** in the top nav with unread count badge
- **@Mention alerts** — Get notified when someone tags you in a thread or comment
- **Reply notifications** — Know when someone responds to your thread
- **Reaction notifications** — See when people react to your posts
- **Role change alerts** — Notified when an admin updates your role
- **Mark as read** — Individual or bulk "mark all read" with one click

### ✍️ Markdown Editor

- **Full formatting toolbar** — Bold, italic, headings, links, images, code blocks, lists, quotes
- **Live preview** toggle to see your post rendered in real-time
- **Keyboard shortcuts** — `Ctrl+B` for bold, `Ctrl+I` for italic, `Ctrl+K` for links
- **Syntax highlighting** for code blocks in 20+ languages
- **Collapsible formatting guide** with syntax examples and tips
- **@mention button** for quick user tagging

### 🟢 Online Presence

- **Green dot indicator** on user avatars when they're currently online
- **"Last seen" timestamps** — Know when someone was last active
- Visible in threads, comments, admin panel, and the user list

### 👑 Admin Panel

- **Overview dashboard** — User count, repo count, thread count, pending reports at a glance
- **User management** — Search users, assign roles via dropdown, suspend/unsuspend accounts
- **Role management** — Create, edit, and delete custom roles with live badge preview
- **Category management** — Add/edit/delete forum categories, set icons & colors, assign role-based access
- **Content reports** — Review, resolve, or dismiss user-submitted reports
- **Audit logging** — Every sensitive action tracked with user, IP, and timestamp

### 🔒 Security & Privacy

- **Invite-only** — No random signups; new users need a valid invite code
- **Argon2id password hashing** — The strongest password hashing algorithm available
- **HttpOnly session cookies** — No JavaScript access to auth tokens, 30-day sessions
- **CSRF protection** — Origin-checked on every mutation
- **Rate limiting** — Anti-brute-force on login (5/5min), signup (3/hr), and uploads (10/min)
- **Content Security Policy** headers to block XSS
- **Markdown sanitization** — Scripts, event handlers, and `javascript:` URIs are stripped
- **Path traversal protection** on all file uploads
- **Full audit trail** — Every login, role change, suspension, and admin action is logged

### 🎨 Midnight Theme

- **Deep dark UI** — `#0a0e1a` background with blue-accented surfaces
- Card-based, spacious layouts that are easy on the eyes
- Smooth animations and hover transitions throughout
- Fully responsive — works on desktop, tablet, and mobile
- Syntax-highlighted code blocks that blend with the theme

## Getting Started

KDONTOP is **invite-only**. If you've received an invite code, head to the signup page, enter your code, and you're in.

Once inside:

1. **Customize your profile** — Set a display name and avatar in Settings
2. **Explore the forum** — Check Announcements for the latest updates
3. **Browse repositories** — See what the community is building
4. **Start a thread** — Ask a question, share an idea, or just say hi in Off-topic
5. **Create a repository** — Share your first project with the community

## Built With

KDONTOP is built on a modern, production-ready stack:

- **Next.js 14** — Full-stack React framework
- **PostgreSQL** — Rock-solid relational database
- **Tailwind CSS** — Utility-first styling with a custom midnight theme
- **Prisma ORM** — Type-safe database access
- **Argon2id** — Industry-leading password security
- **Docker** — One-command deployment
